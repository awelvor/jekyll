Hello, world!

Jekyll est un générateur de sites statiques simple, adapté aux blogs personnels, aux sites de portfolio et aux documentations. Il prend votre contenu écrit en Markdown, applique des mises en page Liquid, et génère un site web statique complet, prêt à être déployé.

Vous pouvez personnaliser l'apparence en modifiant les fichiers CSS ou Sass de votre projet. Si vous utilisez Bootstrap, vous pouvez surcharger les variables Sass par défaut de Bootstrap ou simplement ajouter vos propres règles CSS pour cibler des classes spécifiques.

Les images, fichiers JavaScript et autres ressources statiques sont généralement placés dans le dossier **/assets** (ou **/public**). Vous devez ensuite les référencer dans votre code en utilisant le filtre \{\{ 'chemin/vers/votre/fichier.jpg' | relative_url \}\}.

Oui, absolument. Le moteur Liquid de Jekyll traitera le contenu Markdown s'il est placé dans un fichier .html qui utilise l'instruction \{\{ content \}\}. Dans le cas d'un simple Accordéon HTML/Markdown, vous pouvez écrire votre texte en Markdown (par exemple, **texte en gras**) directement dans le accordion-body si l'Accordéon est dans un fichier Markdown principal.
